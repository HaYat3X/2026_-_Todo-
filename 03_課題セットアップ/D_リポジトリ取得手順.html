事前確認

今回の前提
  • Git が使えること
  • Node.js と npm が使えること
  • PowerShell が起動できること
  • 講師から指示があったリポジトリURL が分かること
当日に必要なもの
  • 講師から指示があった GitHub リポジトリURL
  • 必要であれば GitHub へのログイン状態
  • インターネット接続

GitHub から取得する手順

1
PowerShell を開く

通常の PowerShell で問題ありません。

2
作業したい場所へ移動する

リポジトリを置きたい場所へ移動します。ここでは例として Desktop を使います。

cd $HOME\Desktop
3
講師から指示があったリポジトリをクローンする

下の <講師から指示があったリポジトリURL> は、実際に案内された URL に置き換えてください。 

git clone <講師から指示があったリポジトリURL>
4
クローンしたフォルダへ移動する

クローン後に作成されたフォルダ名へ移動します。

cd <クローンしてできたフォルダ名>
5
依存関係を入れる

研修で使うアプリが Node.js ベースの場合は、依存関係をインストールします。

npm install
6
開発サーバーを起動する

研修で起動確認まで行う場合は、次のコマンドを実行します。

npm run dev
成功の目安 PowerShell にローカル URL が表示され、ブラウザで画面が開ければ成功です。
7
ここまでのコマンドをまとめて確認する
cd $HOME\Desktop
git clone <講師から指示があったリポジトリURL>
cd <クローンしてできたフォルダ名>
npm install
npm run dev

トラブル対応

git が見つからない

  • Git が事前導入されているか確認する
  • PowerShell を閉じて開き直す
  • それでも解消しなければ研修担当者へ相談する

リポジトリにアクセスできない

  • URL が正しいか確認する
  • 自分のチームのリポジトリか確認する
  • GitHub にログインが必要か確認する
  • 研修担当者へ共有する
わからないことはまずチームで。それでも詰まったらチューターへ。
