Visual Studio Code インストール手順書

VS Code のセットアップと日本語化拡張機能の導入手順
対象：新入社員 OS：Windows 11 想定所要時間：5〜10分

01 セットアップウィザードの進行

インストーラ（exeファイル）を実行し、画面の指示に従ってインストールを進めます。最初の画面で「同意する」を選択した後は、基本的にデフォルトの設定のまま進めて問題ありません。

1
使用許諾契約書の同意

「同意する(A)」を選択し、「次へ(N)」をクリックします。

2
インストール先の指定

デフォルトのまま変更せず、「次へ(N)」をクリックします。

3
スタートメニューフォルダーの指定

デフォルトのまま変更せず、「次へ(N)」をクリックします。

4
追加タスクの選択

「サポートされているファイルの種類のエディターとして、Codeを登録する」と「PATH への追加（再起動後に使用可能）」にチェックが入っていることを確認し、「次へ(N)」をクリックします。

5
インストール準備完了

設定内容を確認し、「インストール(I)」をクリックしてインストールを開始します。

6
セットアップウィザードの完了

インストールが完了したら、「Visual Studio Code を実行する」にチェックが入っていることを確認して「完了(F)」をクリックします。

成功の目安 VS Code のウィンドウが自動的に起動すれば完了です。

02 拡張機能のインストール

VS Code のインストールが完了してアプリが起動したら、メニュー等を日本語で表示するための拡張機能をインストールします。

1
拡張機能画面を開く

画面左側にあるメニューから、四角形（□）が組み合わさったアイコンをクリックします。

拡張機能アイコン
2
拡張機能の検索

左上に表示される検索欄に ja と入力します。

3
日本語化パックのインストール

検索結果に表示される「Japanese Language Pack for Visual Studio Code」を選択し、インストールします。

Japanese Language Pack
インストール後

拡張機能インストール後、右下に再起動を促すダイアログが表示されます。「Restart」または「再起動」をクリックすると日本語UIが適用されます。
