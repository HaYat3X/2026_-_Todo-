Node.js インストール手順書

Windows 11 / Node.js / npm ／ 想定所要時間：10〜15分

この手順書で行うこと

使用するもの

Node.js 24 LTS
npm は Node.js に含まれます。

ここではやらないこと

GitHub リポジトリのクローン
研修当日に行います。

確認すること

PowerShell でバージョンが表示されること
node -v と npm -v が通れば成功。

事前確認

使用する PC
  • Windows 11 が起動していること
  • インターネットへ接続できること
  • ブラウザが使えること
  • PowerShell が起動できること
今回の前提
  • Node.js はまだ入っていない想定です
  • npm を別で入れる必要はありません
  • PowerShell は通常の起動で問題ありません
  • 必要な場合のみインストーラーが管理者権限の確認を表示します

インストール手順

1
Node.js のダウンロードページを開く

Node.js の公式ダウンロードページを開き、LTS の版を使用します。 2026年4月時点では Node.js 24.14.1 が最新 LTS です。

Node.js 公式ダウンロードページ Node.js 24.14.1 (LTS) リリース情報
今回ダウンロードするもの： 研修 PC は共通で 64bit のため Windows Installer (.msi) / x64 を選んでください。
確認する場合は [設定] > [システム] > [バージョン情報] で「システムの種類」に「64 ビット オペレーティング システム、x64 ベース プロセッサ」と表示されているか確認してください。
2
Node.js をインストールする
  1. ダウンロードした .msi ファイルをダブルクリックします。
  2. 画面の案内に従って進めます。基本的には既定値のままで問題ありません。
  3. インストールが完了したら、いったん PowerShell を閉じます。
  4. PowerShell を開き直します。
通常は管理者権限で PowerShell を開く必要はありません。必要な場合はインストーラーの途中で Windows の確認画面が表示されます。
3
Node.js と npm が入ったか確認する

PowerShell を開いて、次のコマンドを実行します。

node -v
npm -v

数字が表示されれば成功です。例: v24.14.1、11.x.x など。

成功の目安 PowerShell にバージョン番号が表示されれば、インストールは完了です。

トラブル対応

node または npm が見つからない

  • PowerShell を閉じて開き直す
  • Node.js のインストールが完了しているか確認する
  • それでも解消しなければ再インストールする

npm.ps1 cannot be loaded ... が出る

PowerShell の実行ポリシーが原因の可能性があります。

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

会社 PC ではポリシーが管理されている場合があります。その場合は担当者へ確認してください。
